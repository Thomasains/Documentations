    PowerShell Commands

    OS Information

    Get-ComputerInfo | Select OsName, WindowsVersion, OsArchitecture

    CPU Details

    Get-CimInstance Win32_Processor | Format-Table Name, NumberOfCores, NumberOfLogicalProcessors

    RAM Modules

    
Get-CimInstance Win32_PhysicalMemory |
Select Manufacturer, Speed,
@{Name="Capacity(GB)";Expression={[math]::Round($_.Capacity/1GB,2)}} |
Format-Table -AutoSize

    Storage Devices

    
Get-PhysicalDisk |
Select Model, MediaType, BusType,
@{Name="Size(GB)";Expression={[int]($_.Size/1GB)}} |
Format-Table -AutoSize
